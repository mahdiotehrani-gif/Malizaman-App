راهنمای نصب و اجرای اپ مالی‌زمان (نسخهٔ آماده برای ساخت در Android Studio)

پیش‌نیازها:
- کامپیوتر با Android Studio نصب‌شده (نسخهٔ 2022.3 یا بالاتر توصیه می‌شود).
- JDK 17
- یک دستگاه اندروید یا شبیه‌ساز با API >= 24

مراحل:
1. فایل ZIP پروژه را از لینک دانلود کن و از حالت فشرده خارج کن.
2. Android Studio را باز کن و گزینه "Open an existing Android Studio project" را انتخاب کن.
3. مسیر پوشه بازشده را انتخاب کن (پوشه حاوی settings.gradle).
4. اجازه بده Gradle و وابستگی‌ها دانلود شوند.
5. پروژه را Build کن (Build > Make Project).
6. دستگاه یا شبیه‌ساز را متصل کن و Run کن.

پس از نصب روی دستگاه:
- هنگام اولین اجرا از کاربر درخواست مجوزهای READ_SMS و RECEIVE_SMS خواهد شد — حتماً آن‌ها را مجاز کن تا استخراج تراکنش‌ها کار کند.
- بخش‌های دیگر مانند همگام‌سازی با Google Calendar نیاز به پیکربندی اضافی در کنسول Google Cloud و قرار دادن کلیدها/OAuth client دارند.

نکات امنیتی و انتشار:
- اگر قصد انتشار در گوگل پلی داری و از READ_SMS استفاده می‌کنی، گوگل ازت می‌خواهد توجیه و مستندات دقیقی ارائه کنی؛ ممکن است نیاز به تغییر طراحی (مثلاً استفاده از واردسازی خودکار از کاربر) باشد.
- برای حفاظت از داده‌های حساس، از رمزنگاری پایگاه‌داده یا قفل برنامه استفاده کن.
